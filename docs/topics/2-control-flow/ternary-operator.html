Ternary Operator

Conditional one-liners with the ternary operator.

Ternary Operator

Condition ? If true : If False

With if-else

if-else.js 
const age = 20;
let message;

if (age >= 18) {
    message = "Adult";
} else {
    message = "Minor";
}

With ternary operator

ternary.js 
const age1 = 20;
const message1 = age1 >= 18 ? "Adult" : "Minor";
// Is age >= 18? If yes, "Adult". If no, "Minor".

Warning

Never nest the ternary operators, it becomes unreadable

nested-bad.js 
const item = isRaining ? "Umbrella" : isSunny ? "Sunglasses" : "Nothing";
